OpenSnek

A lightweight macOS alternative to Razer Synapse for supported mice.

Download DMG View on GitHub
OpenSnek is 100% free and open source software.
macOS Native Native Swift app with low idle overhead and no extra runtime bloat.
USB + Bluetooth Supports compatible mice over both USB and Bluetooth, depending on the device.
Onboard profiles Supports Synapse-compatible onboard profiles over USB and Bluetooth where the device allows it.
V3 X HyperSpeed Razer Synapse does not support it on macOS; OpenSnek is the only native macOS configuration path for it.

Customize your device without compromises.

OpenSnek surfaces the hardware settings mouse owners actually reach for: DPI stages, active stage, lighting, power, poll rate where available, and button remaps.

macOS 14+ Native Swift Apache 2.0

Core controls

DPI, profiles, and button remaps.

Menu bar DPI control

Quick stage changes stay available without keeping the full window in the way.

Onboard profiles

Manage hardware profiles on supported devices and keep settings on the mouse.

Button remaps

Rebind supported mouse buttons, including keyboard actions and turbo mappings.
